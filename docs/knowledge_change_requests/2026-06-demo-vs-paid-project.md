# Demo y proyecto pagado

## Demo de MovIA

El demo de MovIA permite que una persona pruebe de forma limitada cómo funciona un agente conversacional antes de contratar un proyecto personalizado.

### Características del demo

- El usuario debe registrarse en la aplicación de MovIA.
- Debe entrar al espacio de **Cliente**.
- Puede crear un agente demo.
- El demo tiene un límite de **10 mensajes**.
- Sirve para entender cómo funciona un chatbot y cómo las instrucciones modifican su comportamiento.
- El demo **no requiere depósito**.
- El demo no inicia trabajo personalizado por parte de MovIA.
- El demo no crea un agente de producción.
- El demo no se despliega en el número de WhatsApp del usuario.
- Crear o probar un demo no significa que el usuario ya haya contratado un agente.

## Proyecto de agente personalizado

El proyecto pagado corresponde a la creación de un agente personalizado para el negocio del cliente.

## Diferencia principal

El demo es una prueba limitada de 10 mensajes y no requiere depósito.

El depósito del 50% aplica únicamente al inicio de un proyecto de agente personalizado.
