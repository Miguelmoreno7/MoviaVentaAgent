## WhatsApp Business

Para conectar WhatsApp con MovIA, el cliente debe tener una cuenta de WhatsApp Business.

No puede utilizarse una cuenta personal de WhatsApp.

## Conexión oficial

MovIA utiliza la integración oficial de Meta/Facebook para conectar números de WhatsApp Business.

Esto es importante porque:

- No se usan conexiones no oficiales.
- No se depende de plataformas informales.
- El cliente conecta su WhatsApp Business mediante el flujo oficial.
- Se requieren permisos correctos en Facebook/Meta.

## Requisitos del cliente

El cliente necesita:

- Cuenta de WhatsApp Business.
- Cuenta de Facebook vinculada.
- Permisos suficientes para administrar la cuenta.
- Capacidad de completar el registro de conexión.

## Problemas de conexión

Si el cliente tiene problemas conectando WhatsApp, debe comunicarse con Miguel.

El agente de preventa puede orientar de forma general, pero los problemas específicos de conexión deben manejarse con atención personalizada.

## Integraciones por producto

### MovIA Captura

No incluye integraciones con APIs externas.

Puede acceder a:

- Bases de datos internas.
- Documentos.
- Información estructurada proporcionada por el cliente.
- Base de conocimiento.

No ejecuta acciones externas.

### MovIA Híbrido

Puede incluir hasta 2 acciones o integraciones simples, según el alcance acordado.

Ejemplos:

- Agendar.
- Cotizar.
- Registrar datos.
- Enviar recordatorios.
- Dar seguimiento básico.

Las acciones deben estar definidas antes de construir el agente.

### MovIA Ventas

Aún no disponible.

Puede incluir una lógica comercial más avanzada, pero no debe prometer integraciones complejas sin cotización.

### MovIA Pro Comercial

Aún no disponible.

Puede involucrar un diseño más avanzado, pero las integraciones específicas o features personalizados deben cotizarse aparte.
