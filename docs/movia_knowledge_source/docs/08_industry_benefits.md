# 08 - Beneficios por industria

Este documento sirve como base para RAG y para respuestas comerciales por industria.

## Dentistas y clínicas dentales

### Problemas comunes

- Pacientes preguntan precios, horarios y tratamientos.
- Recepción no siempre responde rápido.
- Muchos mensajes llegan fuera de horario.
- Los pacientes comparan opciones.
- Se pierden citas por falta de seguimiento.

### Cómo ayuda MovIA Captura

- Responde dudas frecuentes.
- Captura nombre, tratamiento de interés y horario preferido.
- Redirige al humano cuando el paciente está listo.
- Ayuda a ordenar los leads antes de que recepción los atienda.

### Cómo ayuda MovIA Híbrido

- Puede agendar citas si esa acción está dentro del alcance.
- Puede enviar recordatorios.
- Puede pedir datos previos del paciente.
- Puede guiar al paciente hacia una cita.

## Médicos y especialistas

### Problemas comunes

- Preguntas repetidas sobre consulta, ubicación, horarios y precios.
- Pacientes que necesitan orientación inicial.
- Leads que se enfrían si no reciben respuesta.

### MovIA Captura

- Ideal para responder información general y filtrar interesados.

### MovIA Híbrido

- Útil si se requiere agendar, confirmar o recordar citas.

## Inmobiliarias

### Problemas comunes

- Preguntas repetidas sobre precio, ubicación, requisitos y disponibilidad.
- Leads llegan por Facebook, Instagram o WhatsApp.
- Muchas personas preguntan pero pocas están calificadas.

### MovIA Captura

- Responde información de propiedades si la base está cargada.
- Califica interesados.
- Redirige al asesor.

### MovIA Híbrido

- Puede registrar visitas, dar seguimiento o ejecutar acciones simples si se define el alcance.

## E-commerce y tiendas

### Problemas comunes

- Preguntas sobre productos, precios, disponibilidad, envíos y garantías.
- Clientes mandan fotos o detalles.
- Se requiere ordenar información antes de atender.

### MovIA Captura

- Útil para responder dudas y capturar leads.

### MovIA Híbrido

- Recomendado si se requiere revisar tickets, garantías, cotizaciones o procesos con proveedores.

## Negocios con campañas de Facebook o WhatsApp Ads

### Problemas comunes

- Muchos leads piden información al mismo tiempo.
- El equipo no responde lo suficientemente rápido.
- Se pierden prospectos por falta de seguimiento inicial.

### MovIA Captura

- Ideal para campañas donde solo se requiere responder, filtrar y redirigir.

### MovIA Híbrido

- Ideal si la campaña busca que el usuario agende, cotice o complete una acción.
