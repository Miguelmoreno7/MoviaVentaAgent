# FAQ de preventa

## ¿Necesito WhatsApp Business?

Sí. Para conectar un número con MovIA se necesita WhatsApp Business, no una cuenta personal.

## ¿Puedo probar antes?

Sí. La app permite crear un demo limitado de 10 mensajes para entender cómo funciona un agente conversacional y cómo cambian sus respuestas con instrucciones.

El demo no requiere depósito, no inicia trabajo personalizado, no crea un agente de producción y no se despliega en el WhatsApp del negocio.

Para construir un agente personalizado sí se requiere el depósito del 50%.

## ¿Qué pasa después de pagar el depósito?

MovIA desbloquea pasos adicionales para subir documentos, ejemplos de conversación e integración de WhatsApp. Después se construye el agente y se entrega una revisión.

## ¿MovIA Híbrido puede interactuar con herramientas externas?

Sí puede realizar hasta 2 acciones operativas acordadas, como agendar, cotizar, registrar información, crear o actualizar un registro, consultar información o enviar información a una herramienta o proceso aprobado.

Si la acción involucra un CRM, ERP, sistema contable, plataforma propietaria, API no revisada, permisos especiales o reglas complejas, MovIA debe revisar técnicamente el proceso antes de confirmar compatibilidad.

## ¿Cuándo pago el restante?

Después de aprobar el agente.

## ¿Hay reembolso?

No. No hay reembolso por ningún motivo.

## ¿Qué pasa si dejo de pagar la mensualidad?

El agente se pausa hasta que se pague la mensualidad.
