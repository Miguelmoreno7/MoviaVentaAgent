# Caso de uso: inmobiliarias

Las inmobiliarias reciben muchas preguntas repetidas sobre precio, ubicación, requisitos, disponibilidad, número de habitaciones y condiciones de renta o venta.

MovIA Captura puede responder dudas sobre propiedades si la información está cargada correctamente y puede filtrar interesados antes de enviarlos al asesor.

MovIA Híbrido puede apoyar cuando el proceso requiere agendar visitas, registrar datos del interesado o dar seguimiento básico.
