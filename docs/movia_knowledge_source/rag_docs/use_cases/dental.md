# Caso de uso: clínicas dentales

Las clínicas dentales suelen recibir preguntas repetidas sobre tratamientos, precios, horarios, ubicación, disponibilidad y formas de contacto.

MovIA Captura puede ayudar cuando la clínica necesita responder dudas frecuentes y capturar información básica del paciente, como nombre, tratamiento de interés y horario preferido.

MovIA Híbrido puede ayudar cuando además se requiere agendar citas, enviar recordatorios o guiar al paciente hacia una acción concreta.

El valor principal para una clínica dental es reducir leads perdidos por falta de respuesta rápida y entregar a recepción prospectos más ordenados.
