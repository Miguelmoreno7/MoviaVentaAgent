# Caso de uso: restaurantes y comida

Restaurantes, pastelerías, negocios de comida o catering pueden recibir preguntas sobre menú, precios, ubicación, horarios, pedidos y disponibilidad.

MovIA Captura ayuda a responder información básica y redirigir al cliente.

MovIA Híbrido puede aplicar si se requiere tomar pedidos, registrar datos o ejecutar una acción definida, siempre que esté dentro del alcance.

Si el flujo requiere enviar o consultar información en una herramienta externa, esa acción puede evaluarse como parte de Híbrido, pero no debe prometerse compatibilidad con una plataforma específica sin revisión técnica.
