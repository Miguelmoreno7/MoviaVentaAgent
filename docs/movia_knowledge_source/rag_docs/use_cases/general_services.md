# Caso de uso: servicios generales y oficios

Negocios de fumigación, carpintería, plomería, limpieza, reparación o servicios a domicilio suelen necesitar capturar datos antes de poder dar una respuesta útil.

MovIA Captura puede pedir información inicial y responder dudas frecuentes.

MovIA Híbrido es mejor cuando se requiere cotizar, registrar solicitud, agendar o dar seguimiento.

Si el negocio necesita crear o actualizar registros, consultar información o enviar datos a una herramienta aprobada, Híbrido puede ser una opción adecuada dentro de las 2 acciones acordadas. La compatibilidad con CRMs, ERPs, sistemas contables, APIs o plataformas propietarias debe revisarse técnicamente antes de confirmarse.
