# Comparación: MovIA vs ManyChat

ManyChat puede ser una buena herramienta para crear automatizaciones y flujos, especialmente cuando el cliente quiere construir sus propios procesos.

MovIA se posiciona diferente: es una solución configurada para el negocio del cliente. El cliente entrega información, documentos, ejemplos y objetivos; MovIA construye y opera el agente.

La diferencia principal es DIY vs done-for-you.

ManyChat puede ser más adecuado si el cliente quiere configurar todo por su cuenta.

MovIA puede ser más adecuado si el cliente quiere que alguien le construya el agente, lo conecte con WhatsApp Business y le dé seguimiento.
