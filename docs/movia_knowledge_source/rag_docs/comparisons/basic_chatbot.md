# Comparación: MovIA vs chatbot básico

Un chatbot básico suele seguir flujos rígidos, botones o respuestas predefinidas.

MovIA busca responder con más contexto usando información del negocio, documentos y reglas. Puede entender preguntas más abiertas y entregar respuestas más naturales.

MovIA Captura sigue limitado a responder y capturar información. No ejecuta acciones.

MovIA Híbrido puede ejecutar hasta 2 acciones operativas acordadas si están pactadas en el alcance. Estas acciones pueden incluir agendar, cotizar, registrar información o interactuar con herramientas aprobadas cuando sea técnicamente viable.
