# Agente de WhatsApp

Un agente de WhatsApp de MovIA responde mensajes dentro del WhatsApp Business del cliente.

Puede usar información del negocio para responder dudas, capturar datos y redirigir a humano.

Dependiendo del producto contratado, puede limitarse a responder o también ejecutar acciones operativas acordadas.

MovIA Captura no ejecuta acciones externas. MovIA Híbrido puede incluir hasta 2 acciones acordadas, como agendar, cotizar, registrar información o interactuar con una herramienta o proceso aprobado cuando el alcance sea claro y técnicamente viable.
